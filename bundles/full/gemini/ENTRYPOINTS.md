# Entrypoints

Use this page as the fastest way into the installed Gemini extension lanes.

## Start Here

- unclear task: use `office-hours`
- end-to-end delivery: use `autopilot`
- bounded multi-agent work: use `team`
- review or release gate setup: use `review-automation`
- founder or company work: use `founder-review` or `operating-review`

## Core Commands

- `office-hours`
  - bound the task and choose the next lane
- `autopilot`
  - plan, execute, QA, validate
- `review`
  - findings-first regression review
- `qa`
  - smallest credible verification pass
- `ship`
  - final sign-off lane
- `retro`
  - compact follow-up after completion

## Advanced Commands

- `team`
  - planner, executor, reviewer, verifier split
- `review-automation`
  - PR and release gate contracts

## Company Commands

- `founder-review`
  - wedge, opportunity, and direction review
- `operating-review`
  - KPI, pipeline, blockers, action review
- `record-work`
  - leave resumable operating records

## Recommended First Move

- `normal` preset: use `office-hours`, then `autopilot`
- `orchestrator` preset: use `team`
- `all-in-one` preset: use `office-hours` to choose `autopilot`, `team`, `founder-review`, or `review-automation`
