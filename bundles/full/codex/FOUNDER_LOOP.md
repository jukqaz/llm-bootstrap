# Founder Loop

Use this lane for founder, product, and wedge decisions.

Preferred sequence:

1. Start from `RALPH_PLAN.md`.
2. Capture the current wedge, customer pain, and evidence.
3. Write the smallest credible decision memo.
4. State what changed since the last founder review.
5. End with the next slice and owner.

Representative artifacts:

- Opportunity Brief
- Wedge Decision
- Founder Brief
