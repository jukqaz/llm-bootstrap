# Ralph Plan

Use this contract when the next step needs an explicit plan artifact.

Required sections:

- Goal
- Requested outcome
- Bounded scope
- Owners
- Connectors and evidence inputs
- Approval boundary
- Verification rule
- Next slice

Rules:

- Keep the plan small enough to execute in one lane.
- Separate assumptions from confirmed evidence.
- Name the connector dependency before using it.
- End with the next concrete action, not a vague summary.
