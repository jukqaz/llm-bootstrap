# ROLE-MAP

현재 curated skill set 기준의 대표 흐름만 정리한다.

## 기본 플레이북 (자주 쓰는 흐름)

- 아이디어 탐색: `idea-harvest` -> `idea-triage` -> `market-scan` -> `mvp-scope-cut` -> `experiment-review`
- 코드 품질 정리: `library-guideline-audit` -> `code-polish-simplify` -> `change-completeness` -> `regression-plan` -> `github-pr-ci`
- 문서/릴리즈: `docs-audit` -> `readme-maintainer` -> `release-docs`
- 인프라/배포: `infra-platform` -> `deploy-checklist` -> `infra-release-runbook`
- 보안 점검: `threat-model` -> `security-review` -> `access-audit`
- 스킬 저장소 유지: `docs-audit` -> `doc-linker` -> `skill-indexer` -> `skill-validator`
