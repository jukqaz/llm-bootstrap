- AGENTS.md: 질문 답변 요약
- PRD.md: 요약/목표/비목표/사용 사례/요구사항/성공 지표/리스크/열린 질문
- TRD.md: 요약/아키텍처/데이터/구현 계획/빌드·실행·테스트/보안/리스크/열린 질문
- README.md: 목적 또는 개요/사용 가이드/문서
- ROLE-MAP.md: 기본 플레이북
- CHANGELOG.md(있을 때): 항목 섹션(## ...)
