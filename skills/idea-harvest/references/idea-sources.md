# Idea Sources

## Good places to look

- 내가 자주 불편해하는 일
- 반복해서 수작업하는 일
- 사람들이 자주 묻는 질문이나 불만
- 엑셀, 노션, 메모장, 복붙으로 겨우 버티는 workflow
- 이미 있는 제품이 너무 무겁거나 비싼 문제

## Prompting questions

- 이 문제는 누가 매주 반복해서 겪는가
- 지금 어떤 우회 방법으로 버티고 있는가
- 불편은 검색, 정리, 자동화, 공유, 추적 중 어디에 가까운가
- 너무 큰 시장 대신 작은 niche에서 먼저 풀 수 있는가
- 내가 1주 안에 프로토타입을 만들 수 있는가

## Avoid

- 아이디어 이름만 멋지고 문제 정의가 없는 것
- 너무 넓어서 첫 사용자와 핵심 플로우가 보이지 않는 것
- 이미 있는 대형 제품을 이유 없이 통째로 복제하려는 것
