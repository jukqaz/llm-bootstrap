# Idea Output Format

## Suggested shape

- Theme: <theme name>
- Candidate: <short working title>
- User: <who has this problem>
- Problem: <what is painful or slow>
- Why now: <why this matters now>
- Existing workaround: <how they solve it today>
- Next question: <what to validate next>

## Shortlist rule

- keep: 바로 더 보고 싶은 후보
- maybe: 아직 애매하지만 보류 가치가 있는 후보
- drop: 지금 기준으로 너무 넓거나 약한 후보
