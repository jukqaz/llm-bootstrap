# 문서 스타일 점검

## Overview

문서 언어, 파일명 대소문자, 링크 규칙을 점검하라. 수정은 사용자 승인 후에만 진행하라.

## Workflow

### 1) 대상 문서 확인
- `AGENTS.md`, `README.md`, `PRD.md`, `TRD.md`, `CHANGELOG.md`를 확인하라.
- 누락 문서는 목록에 기록하라.

### 2) 스타일 점검
- 문서 언어가 한국어인지 확인하라.
- 파일명 대소문자가 규칙과 일치하는지 확인하라.
- 링크 대상 파일명과 실제 파일명이 일치하는지 확인하라.

### 3) 결과 리포트
- 위반 항목을 우선순위로 정리하라.
- 수정 제안은 하되, 승인 없이는 수정하지 말아라.

## Output checklist

- 누락 문서 목록이 포함된다.
- 언어/대소문자/링크 위반이 정리되어 있다.
