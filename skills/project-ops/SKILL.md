# 프로젝트/운영

## Overview

프로젝트/운영 역할에서 필요한 산출물과 작업 흐름을 정리하라. 한국어로 작성하고 문서/작업 결과를 간결하게 요약하라.

## Workflow

### 1) 컨텍스트 수집
- AGENTS.md, README.md, PRD.md, TRD.md를 확인하라.
- 목표, 범위, 제약, 일정 정보를 정리하라.
- 정보가 부족하면 먼저 질문하라.

### 2) 산출물 정의
- 필요한 산출물을 작업 계획, 일정, 리스크 로그, 진행 상태 기록 기준으로 정리하라.
- 산출물 우선순위를 정하고 범위를 명확히 하라.

### 3) 세부 스킬 연계
- 필요 시 다음 스킬을 사용하라: pre-work-plan, plan-archive
- 외부 도구나 MCP 설정이 필요하면 먼저 확인하라.

### 4) 정리 및 공유
- 결과를 핵심 요약과 함께 공유하라.
- 관련 문서/링크를 갱신하라.
