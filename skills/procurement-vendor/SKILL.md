# 조달/벤더

## Overview

벤더 관리와 구매 절차를 문서화하라.

## Workflow

### 1) 컨텍스트 수집
- 구매/외주 범위와 예산을 확인하라.

### 2) 산출물 정의
- `references/vendor-brief.md`로 요약을 작성하라.
- `references/procurement-checklist.md`를 사용하라.

### 3) 공유
- 승인/검토 흐름을 명확히 하라.

## Output checklist

- 범위/예산이 명확하다.
- 검토/승인 흐름이 포함된다.
