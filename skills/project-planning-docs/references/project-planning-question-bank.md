# 기획 질문 목록

필수 질문부터 답변을 받은 뒤 문서 작성을 시작하라.

## 필수 (Need to know)
1) 프로젝트 한 줄 목표는?
2) 대상 사용자/사용 사례는?
3) 범위 포함/제외 항목은?
4) 성공 지표 또는 수용 기준은?
5) 일정/마감 또는 제약(기술, 예산, 일정)은?

## 기술 필수 (Need to know)
1) 예상 아키텍처 또는 주요 컴포넌트는?
2) 데이터 저장 방식/모델은?
3) 외부 연동 또는 API가 있는가?
4) 실행/테스트 환경 또는 필수 도구는?
5) 소스관리 도구와 워크플로우는? (예: git, jj)

## 선택 (Nice to know)
1) UX/사용자 흐름 요약이 필요한가?
2) 리스크와 대응 방안이 있는가?
3) 마일스톤 구성이 필요한가?

## 응답 가이드
- 불확실한 항목은 "미정"으로 표시하고 가정을 명시하라.
- 답변은 번호 순서대로 짧게 정리하라.
