# 로컬라이징/국제화

## Overview

번역/현지화 규칙을 정리하라.

## Workflow

### 1) 컨텍스트 수집
- 대상 언어/지역을 확인하라.

### 2) 산출물 정의
- `references/i18n-guide.md`로 가이드를 작성하라.
- `references/locale-checklist.md`로 점검하라.

### 3) 공유
- 변경 범위와 책임을 명확히 하라.

## Output checklist

- 언어/지역 범위가 명확하다.
- 규칙/용어가 정리되었다.
