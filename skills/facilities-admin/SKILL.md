# 시설/총무

## Overview

시설 운영과 자산 관리 문서를 정리하라.

범위: 시설 운영/자산
비범위: IT 계정/권한(it-internal-systems), 조달/벤더(procurement-vendor)

## Workflow

### 1) 컨텍스트 수집
- 대상 시설/자산 범위를 확인하라.

### 2) 산출물 정의
- `references/facilities-checklist.md`와 `references/facilities-asset-log.md`를 사용하라.

### 3) 공유
- 유지보수/요청 경로를 명확히 하라.

## Output checklist

- 시설 범위와 절차가 명확하다.
- 자산 기록이 포함된다.
