# 계획 아카이브

## Overview

완료된 계획을 `plans/archive/`로 이동하고 `PLAN.md` 인덱스를 갱신하라. 승인 없이 파일을 이동하지 말아라.

## Workflow

### 1) 대상 확인
- `PLAN.md`와 `plans/` 디렉터리를 확인하라.
- Done/Abandoned 상태의 항목을 찾으라.

### 2) 아카이브 기준
- 최신 5~10개 항목은 `PLAN.md`에 유지하라.
- 그 외 완료 항목은 `plans/archive/`로 이동하라.

### 3) 이동 및 링크 갱신
- 이동 전 승인받아라.
- `PLAN.md`의 링크를 업데이트하라.

### 4) 결과 요약
- 이동된 파일 목록과 인덱스 변경을 요약하라.

## Output checklist

- 최신 항목이 인덱스에 유지된다.
- 아카이브 경로가 정확하다.
- 승인 후에만 이동했다.
