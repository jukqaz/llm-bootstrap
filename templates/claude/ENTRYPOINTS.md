# Entrypoints

Use this page as the fastest way into the installed Claude lanes.

## Start Here

- unclear task: open `OFFICE_HOURS.md`
- end-to-end delivery: open `AUTOPILOT.md`
- bounded multi-agent work: open `TEAM.md`
- review or release gate setup: open `REVIEW_AUTOMATION.md`
- incident or repeated failure: open `INVESTIGATE.md`
- founder or company work: open `FOUNDER_LOOP.md` and `OPERATING_REVIEW.md`

## Core Lanes

- `office-hours`
  - bound the task and pick the next lane
- `autopilot`
  - plan, execute, QA, validate
- `review`
  - findings-first regression review
- `qa`
  - smallest credible verification pass
- `ship`
  - final sign-off lane
- `retro`
  - compact follow-up after completion

## Advanced Lanes

- `team`
  - planner, executor, reviewer, verifier split
- `review-automation`
  - PR and release gate contracts

## Company Lanes

- `founder-loop`
  - wedge, positioning, next-bet decisions
- `operating-review`
  - KPI, pipeline, blockers, action review
- `record-work`
  - leave resumable operating records

## Recommended First Move

- `normal` preset: start with `office-hours`, then `autopilot`
- `orchestrator` preset: start with `team`
- `all-in-one` preset: use `office-hours` to choose `autopilot`, `team`, `founder-loop`, or `review-automation`
