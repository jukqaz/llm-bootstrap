# StackPilot Dev Kit

`stackpilot-dev-kit` is the addon bundle that sits on top of the `StackPilot`
core.

Repository layout:

- `codex/`
  - Codex plugin source assets copied into `~/.codex/plugins/stackpilot-dev-kit`
- `gemini/`
  - Gemini extension source assets copied into `~/.gemini/extensions/stackpilot-dev`
- `claude/`
  - Claude addon skills copied into `~/.claude/skills/*`
- `bundles/`
  - addon-only bundle assets that are rendered when a pack requires them
